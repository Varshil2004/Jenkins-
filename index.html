⚙️ Jenkins Installation & Dashboard Overview

🖥️ Step 1: Install Java (Required)

Jenkins requires Java to run.

sudo apt update
sudo apt install openjdk-17-jdk -y
java -version
⚠️ Warning: Jenkins supports Java 11 and 17. Do NOT use Java 8 or Java 21+.

📥 Step 2: Install Jenkins

Run these commands in your terminal (for Ubuntu/Debian):

wget -q -O - https://pkg.jenkins.io/debian-stable/jenkins.io.key | sudo apt-key add -
sudo sh -c 'echo deb https://pkg.jenkins.io/debian-stable binary/ > /etc/apt/sources.list.d/jenkins.list'
sudo apt update
sudo apt install jenkins -y

🚀 Step 3: Start Jenkins Service

sudo systemctl start jenkins
sudo systemctl enable jenkins
sudo systemctl status jenkins

🔓 Step 4: Unlock Jenkins

Visit http://localhost:8080 in your browser.

To get the admin password, run:

sudo cat /var/lib/jenkins/secrets/initialAdminPassword

🛠 Step 5: Install Plugins & Create User

  • Click "Install Suggested Plugins"
  • Create your first admin user
✅ Tip: If port 8080 is in use, change it in /etc/default/jenkins

🌐 Optional: Expose Jenkins Using Ngrok

1. Download Ngrok and install it.
2. Authenticate with Ngrok: 
ngrok config add-authtoken YOUR_TOKEN
3. Start Jenkins tunnel: 
ngrok http 8080
4. Use the generated URL like: https://abc123.ngrok.io

🔧 How Jenkins Works (Simplified)

  • 👨‍💻 Developers push code to GitHub
  • 🔁 Jenkins detects changes via Webhook
  • ⚙️ Jenkins builds and tests the project
  • 🚀 If successful, Jenkins deploys it (optional)

📊 Jenkins Dashboard Overview

  • New Item: Create a Freestyle or Pipeline job
  • Build Now: Run a job manually
  • Configure: Set job options (Git URL, build script, etc.)
  • Build History: Shows logs, test results, status

✅ Final Checklist

  • ✔ Jenkins running at http://localhost:8080
  • 🔗 Ngrok public URL (optional)
  • 🧪 Ready to integrate CI/CD with GitHub

💡 Best Practices

  • Use pipelines over freestyle for scalability
  • Secure Jenkins with authentication + HTTPS
  • Use GitHub Webhooks for automatic builds
🛡️ Security: Install plugins like Role Strategy, backup jobs, and limit anonymous access.


🎉 You're now ready to build CI/CD pipelines with Jenkins!
